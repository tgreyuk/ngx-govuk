# Accordion

The accordion component lets users show and hide sections of related content on a page.

## GOV.UK Docs

[https://design-system.service.gov.uk/components/accordion/](https://design-system.service.gov.uk/components/accordion/)

## Usage

```ts
import { NgxGovukAccordionComponent } from 'ngx-govuk/accordion';

@Component({
  standalone: true,
  imports: [NgxGovukAccordionComponent],
  selector: 'my-component',
  template: `<ngx-govuk-accordion></ngx-govuk-accordion>`,
})
```
